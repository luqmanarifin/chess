# chess

A simple chess.

Note: Apabila gambar muncul buruk di terminal Anda, mungkin terminal Anda terlalu kecil.

Screenshot:

![alt text](http://i.imgur.com/IXqpGNY.png "Screenshot")

To compile:
```javascript
make compile
```

To run:
```javascript
make run
```

Symbol in-game:

H: King

O: Queen

X: Bishop

L: Knight

+: Rook
